## 1. Discord Bot Privacy Policy

This bot collects only the minimum data required to operate its features.

### Data Collected

- Discord usernames and user IDs
- Messages submitted through ticket systems

### Purpose

- To manage support tickets and respond to user requests
- To maintain moderation and server organization

### Data Access

Data is accessible only to authorized staff handling tickets and moderation.

### Data Retention

Data is stored only as long as necessary for ticket resolution and moderation logs.

### Data Sharing

No data is sold or shared with third parties.

### User Rights

You may request deletion of your data by contacting server staff.

> **Notice:** By using the bot, you agree to this data handling.

## 2. Minecraft Server Privacy Policy

This server collects limited technical and gameplay-related data required for operation and moderation.

### Data Collected

- IP addresses (for security, GeoIP features, and IP bans)
- Minecraft UUIDs and usernames
- Private messages (/msg, /tell, /w) which may be logged

### Purpose

- To maintain server security and prevent abuse
- To enforce rules and investigate harassment or cyberbullying
- To provide features such as GeoIP-based functionality

### Data Access

Data is accessible only to authorized administrators and moderators.

### Data Retention

Logs and data are retained only as long as necessary for moderation and server operation.

### Data Sharing

No data is sold or shared with external parties unless required by law.

### Third-Party Integrations

The server uses plugins such as EssentialsX and DiscordSRV, which may process data as part of their functionality.

### User Rights

You may request information about or deletion of your data by contacting server staff.

> **Notice:** By joining the server, you consent to this data collection and usage.
